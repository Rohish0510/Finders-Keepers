# FindersKeepers - User Stories

## Authentication
- As a user, I want to register an account so that I can use the system.
- As a user, I want to login securely using JWT.
- As a user, I want protected routes so unauthorized users cannot access resources.

## Item Management
- As a user, I want to post lost items.
- As a user, I want to post found items.
- As a user, I want to view all open items.

## Claim System
- As a user, I want to claim an item.
- As an item owner, I want to approve or reject claims.
- As a system, I want to prevent users from claiming their own items.

## Deployment
- As a developer, I want the app to run inside Docker.
- As a developer, I want CI/CD to build and push images automatically.
